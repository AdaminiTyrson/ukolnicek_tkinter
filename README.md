# Úkolníček – Task Manager (Python Tkinter)

Jednoduchá desktopová aplikace pro správu úkolů vytvořená v Pythonu s využitím Tkinteru. Cílem projektu bylo vytvořit přehlednou aplikaci a procvičit práci s GUI a řízením stavu aplikace.

Funkce
- přidání nového úkolu
- odstranění úkolu
- zobrazení seznamu úkolů
- práce s uživatelskými vstupy

Technologie
- Python
- Tkinter

Co projekt ukazuje
- návrh a tvorbu grafického rozhraní
- event-driven programování (reakce na akce uživatele)
- práci se stavem aplikace a daty v paměti
- strukturování kódu pro lepší čitelnost
